### 트리(Tree)의 순회(Traverse) 알고리즘

![](/algorithm/binary_tree.png)  

- 전위 순회(Pre-Order Traverse)  
  - 1 -> 2 -> 4 -> 5 -> 3 -> 6 -> 7  

- 중위 순회(In-Order Traverse)  
  - 4 -> 2 -> 5 -> 1 -> 6 -> 3 -> 7  

- 후위 순회(Post-Order Traverse)  
  - 4 -> 5 -> 2 -> 6 -> 7 -> 3 -> 1  

- 단계 순위 순회(Level-Order Traverse)  
  - 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7
